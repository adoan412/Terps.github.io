# Terpene Search Tool 🌿

A lightweight, browser-based terpenesearch tool for budtenders and cannabis enthusiasts to quickly look up terpene profiles and their effects.

## What This Is

The Terpene Search Tool is a single-file HTML application that provides instant access to comprehensive terpene data — no installation, no API calls, just open your browser and start searching.

Built by Tony (@adoan412) for the dispensary community, because everyone deserves clear information about what's in their products.

## Features

- **Instant search** across 50+ terpenes
- **Detailed profiles** including scent notes, effects, and common strain associations
- **Offline-capable** — works without internet once loaded
- **Zero dependencies** — pure HTML/CSS/JS, runs in any browser
- **Privacy-first** — no tracking, no analytics, your data stays with you

## Why Open Source?

Because cannabis education should be accessible to everyone. This tool was built for budtenders at my dispensary, and I want others to benefit from it too. Feel free to:

- Use it in your own shop
- Modify it for your needs
- Share it freely with colleagues
- Improve the data or UI (pull requests welcome!)

## Installation

1. Download `terpenesearch.html` from this repo
2. Open it in any modern web browser (Chrome, Firefox, Safari, Edge)
3. Start searching! No setup required.
